    requireSpacesInForStatement

    Requires spaces inbetween for statement.

    Type: Boolean

    Value: true to requires spaces inbetween for statement.

    Example

    "requireSpacesInForStatement": true
    Valid
    for(var i = 0; i<l; i++) {
    x++;
}
    Invalid
    for(var i = 0;i<l;i++) {
    x++;
}

    for(var i = 0; i<l;i++) {
    x++;
}

    for(var i = 0;i<l; i++) {
    x++;
}
