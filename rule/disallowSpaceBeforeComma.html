    disallowSpaceBeforeComma

    Disallows spaces before commas

    Types: Boolean

    Values: true to disallow any spaces before any comma

    Example

    "disallowSpaceBeforeComma": true
    Valid
    var a, b;
    Invalid
    var a ,b;
