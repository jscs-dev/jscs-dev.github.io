    disallowSpaceBeforeKeywords

    Disallows space before keyword.

    Types: Array or Boolean

    Values: Array of quoted keywords or true to disallow spaces before all possible keywords.

    Example

    "disallowSpaceBeforeKeywords": [
    "else",
    "catch"
]
    Valid
    }else {
    y--;
}
    Invalid
    } else {
    y--;
}
