    disallowPaddingNewlinesInBlocks

    Disallows blocks from beginning or ending with 2 newlines.

    Type: Boolean

    Value: true validates all non-empty blocks.

    Example

    "disallowPaddingNewlinesInBlocks": true
    Valid
    if (true) {
    doSomething();
}
