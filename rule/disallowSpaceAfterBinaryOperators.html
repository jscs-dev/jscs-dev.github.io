    disallowSpaceAfterBinaryOperators

    Requires sticking binary operators to the right.

    Types: Array or Boolean

    Values: Array of quoted operators or true to disallow space after all possible binary operators

    Example

    "disallowSpaceAfterBinaryOperators": [
    "=",
    ",",
    "+",
    "-",
    "/",
    "*",
    "==",
    "===",
    "!=",
    "!=="
    // etc
]
    Valid
    x +y;
    Invalid
    x+ y;
