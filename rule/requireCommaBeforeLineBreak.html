    requireCommaBeforeLineBreak

    Requires commas as last token on a line in lists.

    Type: Boolean

    Value: true

    JSHint: laxcomma

    Example

    "requireCommaBeforeLineBreak": true
    Valid
    var x = {
    one: 1,
    two: 2
};
var y = { three: 3, four: 4};
    Invalid
    var x = {
    one: 1
    , two: 2
};
