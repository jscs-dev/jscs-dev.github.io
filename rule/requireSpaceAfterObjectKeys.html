    requireSpaceAfterObjectKeys

    Requires space after object keys.

    Type: Boolean

    Value: true

    Example

    "requireSpaceAfterObjectKeys": true
    Valid
    var x = {a : 1};
    Invalid
    var x = {a: 1};
