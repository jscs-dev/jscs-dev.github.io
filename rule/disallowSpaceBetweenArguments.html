    disallowSpaceBetweenArguments

    Ensure there are no spaces after argument separators in call expressions.

    Type: Boolean

    Value: true

    Example

    "disallowSpaceBetweenArguments": true
    Valid
    a(b,c);
    Invalid
    a(b, c);
