    disallowSpaceBeforeSemicolon

    Disallows spaces before semicolons.

    Types: Boolean

    Values: true to disallow any spaces before any semicolon.

    Example

    "disallowSpaceBeforeSemicolon": true
    Valid
    var a = 1;
    Invalid
    var a = 1 ;
