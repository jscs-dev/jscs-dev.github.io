    disallowTrailingComma

    Disallows an extra comma following the final element of an array or object literal.

    Type: Boolean

    Value: true

    JSHint: es3

    Example

    "disallowTrailingComma": true
    Valid
    var foo = [1, 2, 3];
var bar = {a: "a", b: "b"}
    Invalid
    var foo = [1, 2, 3, ];
var bar = {a: "a", b: "b", }
