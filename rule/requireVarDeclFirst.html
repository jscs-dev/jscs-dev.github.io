    requireVarDeclFirst

    Requires var declaration to be on the top of an enclosing scope

    Types: Boolean

    Values:

    • true specifies that var declarations must occur the top of a function scope.

    Example

    "requireVarDeclFirst": true
    Valid for mode true

    y = 2;
