    requireSpaceAfterKeywords

    Requires space after keyword.

    Types: Array or Boolean

    Values: Array of quoted keywords or true to require all of the keywords below to have a space afterward.

    Example

    "requireSpaceAfterKeywords": [
    "do",
    "for",
    "if",
    "else",
    "switch",
    "case",
    "try",
    "catch",
    "void",
    "while",
    "with",
    "return",
    "typeof",
    "function"
]
    Valid
    if (x) {
    x++;
}
    Invalid
    if(x) {
    x++;
}
