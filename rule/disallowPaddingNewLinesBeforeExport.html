    disallowPaddingNewLinesBeforeExport

    Disallows newline before module.exports

    Type: Boolean

    Value: true

    Example

    "disallowPaddingNewLinesBeforeExport": true
    Valid
    var a = 2;
module.exports = a;
    Invalid
    var a = 2;

module.exports = a;
